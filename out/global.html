zip(list1, list2) → {array/object}

Takes two arrays or one object and returns a new, multidimensional, zipped array or object. A zipped object will have subarrays of [key, value] for each key, value pairing. The zipped array will have subarrays of length two with items from each corresponding position in each array. If the arrays are of unequal length, the result will be zipped based on the length of the shortest array (the remaining items of the longer array will be ignored).
